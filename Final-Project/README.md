# 📚 Final Project: BookTracker

BookTracker is a web-based application designed to help users organize their personal library and track their reading progress. This project serves as the capstone for my journey through **Harvard's CS50x**

## 💻 Tech Stack
- **Backend:** Python & Flask
- **Database:** SQLite (SQL)
- **Frontend:** HTML & CSS

## 🎯 Key Features
- **Library Management:** Users can view a complete list of books with their respective authors and reading statuses.
- **Dynamic Updates:** Easily add new books with statuses like "Want to Read", "Reading", or "Read".
- **Data Persistence:** Integrated SQL database to ensure all records are saved and can be managed (added/deleted) reliably.

## 🧠 Design Decisions
Focused on a **clean and functional user experience**, the application prioritizes core features to ensure reliability. By using Flask for routing and SQL for data management, the project demonstrates a solid understanding of how backend and frontend communicate to solve real-world problems.
